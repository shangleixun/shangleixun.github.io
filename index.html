简明的 GitHub 个人博客搭建教程

Posted on 2019-09-05

心血来潮，终于想依托 Github 创建一个个人博客网站。想到之前看过的教程，感觉头皮发麻，有一丝胆怯，最后硬着头皮做了下来。结果差强人意，不过是可以来写博客了而已。

总体上来说，分三步：

第一步：在 GitHub 上建立专门为博客网站的 repo 。

流程极简单，在 GitHub 个人主页，创建新的 repo ，有一点要注意，如果是第一次创建 *.github.io 类的 repo ，repo name 必须以用户名为前缀，不然会失败。
repo 的名称应该是：username.github.io 。其他默认就好了。

第二步：在电脑上安装 Node.js ，Git ，以及 Hexo 。

使用 macOS 的用户，应该是极方便的，只用安装 Node.js 就可以了。安装方法，我是直接在官网下载稳定包，安装就完事。
